# khairat2u.com
Khairat2u.com adalah sistem pengurusan khairat kematian yang dimiliki oleh KHAIRAT2U.COM berpengkalan di Temerloh, Pahang.

## Pengenalan
Antara modul yang disediakan adalah:
+ Ahli khairat
+ Yuran
+ Pembayaran & Resit
+ Mesej
+ Pengumuman
+ Statistik
+ Tuntutan

## Keperluan teknikal
+ Php
+ Mysql
